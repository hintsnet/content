---
title: "[001] 带有 ag 词根的英语单词"
desc: "看到 ag 就联想到 do 或 act，即联想到 \"做，行动\"。"
format: flashcard/quizlet
---

```
*ag*ent ( 'eidʒənt )	[ ag- 做/办理，-ent 名词后缀，表示人；做事者，"办理人" ] 代理人
agential ( ei'dʒenʃəl )	[ ag- 做/办理，-ent 名词后缀，表示人；做事者，-ial 形容词后缀，...的 ] 代理人的
subagent ( ,sʌb'eidʒənt )	[ sub- 副的，agent 代理人 ] 副代理人
coagent ( kəu'eidʒənt )	[ co- 共同，ag 做/作 -ent 表示人 ] 共事者，合作者
agency ( 'eidʒənsi )	[ ag- 做/作，ency 名词后缀 ] 代理，代理处，机构，作用
coagency ( kəu'eidʒənsi )	[ co- 共同，ag 做/行动，ency 名词后缀] 共事，协作，合作
agenda ( ə'dʒendə )	[ ag- 做，end 名词后缀，a 表示复数；原义为待做的事项 ] 议事日程
agile ( 'ædʒail )	[ ag- 动→活动→灵活 -ile 形容词后缀 ...的 ] 灵活的，敏捷的
agility ( ə'dʒiləti )	[ ag- 动→活动→灵活，-ility 名词后缀，表示抽象名词 ] 灵活，敏捷
agitate ( 'ædʒiteit )	[ ag- 动，-it- 连接字母，-ate 动词后缀：使..."使骚动" ] 鼓动，煽动，搅动，使不安定
agitation ( ,ædʒi'teiʃən )	[ ag- 动，-it- 连接字母，-ation 名词后缀 ] 鼓动，煽动
agitator ( 'ædʒiteitə )	[ ag- 动，-it- 连接字母，-at(e) 动词后缀：使...，-or 表示人 ] 鼓动者，煽动者
agitatress	[ ag- 动，-it- 连接字母，-at(e) 动词后缀，使...，-ress 表示女性 ] 女鼓动家
agitated ( 'ædʒiteitid )	[ ag- 动，-it- 连接字母，-ate 动词后缀，使...，-ed 形容词后缀，...的 ] 不安的
*ag*itating	[ ag- 动，-it- 连接字母，-at(e) 动词后缀，使...，-ing 形容词后缀 ] 使人不安的，进行鼓动的
counteragent ( ,kauntə'reidʒənt )	[ counter- 反，ag 做，作用，-ent 表示物 ] 反作用剂，反抗力
```
